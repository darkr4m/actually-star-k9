## Model: `CalendarEvent`

Stores information about calendar events, potentially linked to both an application user and a Google Calendar event.

| Field Name           | Field Type            | Description                                                                   | Options                                                                                                                                                                                                                       |
| :------------------- | :-------------------- | :---------------------------------------------------------------------------- | :---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `user`               | `ForeignKey(User)`    | The application user associated with this calendar event.                     | `on_delete=models.CASCADE` (If User is deleted, their events are deleted too),<br>`related_name='calendar_events'` (Access events from User object via `user.calendar_events`)                                                |
| `google_event_id`    | `CharField`           | The unique ID assigned by Google Calendar if synced. Used to link records.    | `max_length=255`, `unique=True` (Each Google Event ID must be unique in this table),<br>`null=True` (DB can store NULL), `blank=True` (Form validation allows empty) - needed for events not yet synced or local-only events. |
| `title`              | `CharField`           | The main title or name of the calendar event.                                 | `max_length=255`                                                                                                                                                                                                              |
| `description`        | `TextField`           | Optional longer description or details about the event.                       | `blank=True`, `null=True`                                                                                                                                                                                                     |
| `start_time`         | `DateTimeField`       | The date and time when the event begins (timezone aware recommended).         | -                                                                                                                                                                                                                             |
| `end_time`           | `DateTimeField`       | The date and time when the event ends (timezone aware recommended).           | -                                                                                                                                                                                                                             |
| `address`            | `ForeignKey(Address)` | Optional link to an `Address` record representing the event's location.       | `on_delete=models.SET_NULL` (If Address is deleted, set this field to NULL),<br>`null=True`, `blank=True`,<br>`related_name='events'` (Access events from Address object via `address.events`)                                |
| `created_at`         | `DateTimeField`       | Timestamp automatically set when the event record is first created.           | `auto_now_add=True`                                                                                                                                                                                                           |
| `updated_at`         | `DateTimeField`       | Timestamp automatically updated every time the event record is saved.         | `auto_now=True`                                                                                                                                                                                                               |
| `synced_with_google` | `BooleanField`        | Flag indicating if this event exists in/is synchronized with Google Calendar. | `default=False`                                                                                                                                                                                                               |

**Methods:**

- `__str__(self)`: Returns a string representation of the event, showing the title and start/end times. Example: "Team Meeting (2025-04-11 10:00:00 - 2025-04-11 11:00:00)"
